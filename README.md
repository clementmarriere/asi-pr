# VAE Assignment — Advanced Statistical Inference 2026

Reproduction of **Auto-Encoding Variational Bayes** (Kingma & Welling, 2014) for the ASI course at EURECOM

## Content

- `vae_assignment.ipynb` — main notebook : model implementation, training, and figures
- `report_vae.tex` — LaTeX report 

## How to run

Open `vae_assignment.ipynb` in Google Colab, and run all cells
NB : enable GPU for faster training (around 10 minutes)
